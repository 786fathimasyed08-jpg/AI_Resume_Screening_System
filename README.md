# AI Resume Screening System

## Project Overview
The AI Resume Screening System is a Machine Learning and NLP-based application that analyzes resumes against a job description. It calculates an ATS score, identifies matched and missing skills, and provides AI-powered suggestions to improve the resume.

## Features
- Upload resume in PDF format
- Extract text using PDFPlumber
- Resume preprocessing using NLP
- Keyword extraction
- Skill matching with job description
- ATS score calculation
- Missing skill identification
- AI-powered resume improvement suggestions
- Resume report generation

## Technologies Used
- Python
- Machine Learning
- NLP
- TF-IDF
- Cosine Similarity
- Scikit-learn
- NLTK
- PDFPlumber
- Google Colab

## Project Workflow
1. Upload Resume PDF
2. Extract Resume Text
3. Clean Text
4. Extract Skills
5. Compare with Job Description
6. Calculate ATS Score
7. Find Missing Skills
8. Generate Suggestions
9. Save Resume Report

## Output
- ATS Score
- Matched Skills
- Missing Skills
- Resume Rating
- AI Improvement Suggestions
